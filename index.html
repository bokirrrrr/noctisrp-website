Peraturan Server Roleplay

Selamat datang! Mohon baca dan pahami peraturan berikut ini.

1. General Rules

  • Dilarang menggunakan cheat/modifikasi ilegal.
  • Dilarang melakukan spam di chat OOC/IC.
  • Hormati semua pemain dan admin.

2. Roleplay Rules

  • Dilarang melakukan Powergaming (PG).
  • Dilarang melakukan Metagaming (MG).
  • Dilarang melakukan Random Deathmatch (RDM).
  • Dilarang melakukan Revenge Kill (RK).
  • Wajib menggunakan /me dan /do dalam situasi RP kompleks.

3. Crime & Police RP

  • Dilarang merampok tanpa alasan RP yang kuat.
  • Minimal anggota geng untuk melakukan perampokan: 3 orang.
  • Polisi wajib menanggapi panggilan darurat IC.

4. Admin Rules

  • Dilarang menyamar menjadi admin.
  • Keputusan admin bersifat final.
  • Laporkan penyalahgunaan admin melalui forum/Discord.

5. Hukuman

  • Melanggar peraturan akan diberi sanksi: kick, jail RP, atau ban tergantung pelanggaran.
